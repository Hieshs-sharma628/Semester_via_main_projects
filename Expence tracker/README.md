# Expence Tracker

Simple personal finance tracker written in Python.

## Features

- Add income records
- Add expense records
- View a summary of totals and balance
- Optional Streamlit interface

## Main File

- `Personal_Finance_Expense.py`

## Data Files

The app stores records in local JSON/TXT files when it runs:

- `Personal_Finance_Expense.json`
- `Personal_Finance_Expense.txt`

These files are ignored by Git because they are generated data.

## Run

```bash
python Personal_Finance_Expense.py
```

If Streamlit is installed, the app can also launch the Streamlit UI automatically.
